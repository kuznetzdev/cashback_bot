# Cashback Bot

Прототип телеграм-бота, который помогает пользователям анализировать чеки, строить аналитику расходов и управлять шаблонами кэшбэков. Бот построен вокруг модульной архитектуры и поддерживает следующие сценарии:

- мастер добавления банка с пошаговой навигацией;
- обработка чеков с индикаторами «⏳ Обрабатываю…» и статусами долгих операций (OCR/NLP/БД);
- экран PRO-аналитики, рекомендации, история операций и геймификация;
- управление шаблонами категорий и расширенными настройками уведомлений;
- система расширений для подключения дополнительных рекомендаций.

## Структура проекта

```
project/
  bot.py                # Оркестрация логики бота
  services/             # Сервисы OCR, NLP, БД, аналитики и т.д.
  i18n/translations.py  # Локализация статусов и меню
```

## Локальный запуск тестов

```bash
python -m pytest
```

Тесты моделируют пользовательские сценарии: мастер добавления банка, обработку чека, шаблоны и настройки уведомлений.
