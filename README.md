# cashback_bot

Бот для анализа кешбэков разных банковских приложений.

## Возможности

* Унифицированные экраны и клавиатуры, построенные через `project/services/ui.py`.
* Мастер добавления банка с сохранением данных в `Database`.
* Поддержка ручных шаблонов чеков и рекомендаций от NLP.
* PRO-аналитика, рекомендации, история, геймификация и расширенные уведомления.
* Умный OCR и NLP-пайплайн, сохраняющий транзакции и шаблоны.

## Структура

* `project/bot.py` — основная логика обработки сообщений и навигации.
* `project/services/` — сервисы OCR, NLP, аналитики, расширений и БД.
* `project/i18n/translations.py` — локализация сообщений и статусов.
* `tests/` — модульные тесты сценариев.
* `docs/SCENARIOS.md` — описание пользовательских сценариев.

## Запуск тестов

```bash
pytest
```
